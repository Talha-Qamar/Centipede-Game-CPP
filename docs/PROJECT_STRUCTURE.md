# Project Structure

## Root Files

- `Centipede.cpp` - Main game source file
- `Centipede.vcxproj` - Visual Studio C++ project file
- `Centipede.vcxproj.filters` - Visual Studio filter organization
- `Centipede.vcxproj.user` - Local Visual Studio user settings (not required for sharing)
- `README.md` - Main repository overview

## Asset Folders

- `Textures/` - Game textures and sprites
- `Music/` - Background music
- `Sound Effects/` - Sound effect files
- `fonts/` - Font files

## Output / Build Artifacts

- `x64/` - Local build output (ignored in git)

## Documentation

- `docs/BUILD.md` - Build and run instructions
- `docs/PROJECT_STRUCTURE.md` - Project layout reference
